3. New Project → Empty Project
4. Add PostgreSQL (click + New → Database → PostgreSQL)
5. Add Backend (+ New → Docker Image):
   - Image: trinnode/sentinel-backend:latest

     JWT_SECRET=your-32-char-secret-here
     NODE_ENV=production
     PORT=3001
   - Generate Domain → Copy URL

   - Image: trinnode/sentinel-dashboard:latest

     REACT_APP_API_URL=https://your-backend-url
